Does the thrill of the ticking clock excite you? Do you feel the need to make your code matter more than just day-to-day coding? Let the adrenaline rush get to you as you compete with the best minds to prove your code the finest of the lot. The Debugging Society of NSUT in association with Coding Blocks brings to you Code Like It Matters, an event which promises to be every bit of fun and mind boggling. Give it your all and let the feeling of satisfaction trump all of that at the end as you make your code matter!

Rules of Round 1

  • 1) The first round will be held on Monday , October 22 from 7pm-9pm. It will be an online contest.
  • 2) Top 40% teams will be shortlisted for the onsite round i.e. the second round.
  • 3) Each team can consist of at most 2 members.
  • 4) Registering for the event is mandatory to be eligible for participation and prizes.
  • 5) Contest will be held at https://hack.codingblocks.com. So make sure you have a valid account there using the same E-mail id you provide in the registration form.
  • 6) The link of Round 1 will be mailed to POC of the registered team.

About Round 2

  • 1) It will be held on Thursday, 25th October, 3pm-6 pm.
  • 2) It will be held Onsite at NSUT.
  • 3) This round will also be held at https://hack.codingblocks.com. So make sure you have a valid account there using the same E-mail id you provide in the registration form.
