Does accuracy matter more than speed to you? ASN presents to you BLIND CODE, a coding competition which will help you to challenge your coding skills and accuracy. All you need to do is code some simple programming questions, with the screen covered.
  1. This is an individual onsite event.
  2. Each participant has to write the code for a given problem, with the screen covered.
  3. Participants will judged on the correctness of their solution. Each correct solution will fetch 100 points.
  4. Participants will be allowed to view their code for a maximum of 3 times, each with a penalty of 10 points.
  5. Discussions are not allowed during the event.
